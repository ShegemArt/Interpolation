# Визуализация интерполяции функций

Этот проект представляет собой приложение на C++, которое выполняет интерполяцию математических функций с использованием различных методов (Лагранж, Ньютона, линейный, квадратичный и кубический сплайны) и визуализирует результаты с помощью SFML.


## Возможности

- Интерполяция произвольных функций.
- Выбор типа узлов:
  - Равномерные узлы
  - Узлы Чебышева
- Методы интерполяции:
  - **Метод Лагранжа**
  - **Метод Ньютона**
  - **Линейный сплайн**
  - **Квадратичный сплайн**
  - **Кубический сплайн**
- Визуализация функции и интерполирующих графиков в реальном времени.
- Вывод максимальной ошибки интерполяции.
- Интерактивный консольный интерфейс для выбора интервала, узлов и метода интерполяции.


## Зависимости

- Simple and Fast Multimedia Library для графического отображения.
- Компилятор C++17 или новее.
- Стандартные библиотеки C++: `<iostream>`, `<vector>`, `<cmath>`, `<functional>`.


## Файлы проекта

- `test.cpp` – Основной файл программы, обрабатывает ввод пользователя, выбор узлов, интерполяцию и визуализацию.
- `InterpolationFunctions.h` – Содержит все функции интерполяции и вычисление коэффициентов сплайнов.
- `timesbd.ttf` – Шрифт для отображения текста в окне SFML.


### Целевая функция

```cpp
double Function(double x);    // Функция, которую интерполируем
double dFunction(double x);   // Производная функции
```


### Методы интерполяции

- **Лагранж** – `LagrangeMethod(points, x, func)`

- **Ньютон** – `NewtonMethod(points, x, coef)`, коэффициенты вычисляются `NewtonCoefficients(points, func)`

- **Линейный сплайн** – `SplineLinear(points, x, func)`

- **Квадратичный сплайн** – `SplineQuadratic(points, coef, x, func)`, коэффициенты `ComputeQuadraticSplineCoeffs(points, func, dfunc)`

- **Кубический сплайн** – `SplineCubic(points, y1, y2, h, x, func)`


## Использование

1. Введите интервал `[a, b]` и количество узлов `n`.

2. Выберите тип узлов (равномерные или Чебышёва).

3. Выберите метод интерполяции.

4. Закройте окно визуализации, чтобы выбрать другой метод или интервал.

5. Повторяйте по необходимости или выберите `0` для выхода.


## Примечания

- Программа визуализирует функцию и её интерполяцию:  
  - Красным цветом отображается оригинальная функция.  
  - Синим цветом – интерполирующая.
- Максимальная ошибка интерполяции отображается в левом верхнем углу окна.
- Для корректной работы требуется файл шрифта `timesbd.ttf` в рабочей директории.
